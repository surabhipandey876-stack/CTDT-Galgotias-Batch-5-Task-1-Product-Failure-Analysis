# CTDT-Galgotias-Batch-5-Task-1-Product-Failure-Analysis

## Product

Industrial Machine

## Objective

Analyze machine failure data and identify the major factors responsible for machine failure.

## Features Explored

* Air Temperature
* Process Temperature
* Rotational Speed
* Torque
* Tool Wear

## Failure Types

* Tool Wear Failure (TWF)
* Heat Dissipation Failure (HDF)
* Power Failure (PWF)
* Overstrain Failure (OSF)
* Random Failure (RNF)

## Solutions

* Predictive Maintenance
* Better Cooling System
* Backup Power Supply
* Load Monitoring
* AI-Based Prediction
